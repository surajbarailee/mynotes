CIFS and ATA

CIFS (Common Internet File System)

Common Internet File System (CIFS) is a file-sharing protocol that provides an open and cross-platform mechanism for requesting network server files and services. CIFS is based on the enhanced version of Microsoft’s Server Message Block (SMB) protocol for Internet and intranet file sharing.

CIFS protocol features include:

File access: Supports basic file operations like open, close, read, write and seek.

File change notification: When file or directory contents are modified, the server is notified.

Extended attributes: Supports the addition of non-file system attributes, like author name, content and description.

Batched requests: Facilitates multiple request batching as single messages to reduce round trip latencies.

 

Samba is a free software implementation of the CIFS/SMB networking protocols that allows for the file transfer between Unix and Windows world.

 

 

SCSI, ATA and SATA


Advanced Technology Attachment (ATA) is a standard physical interface for connecting storage devices within a computer. ATA allows hard disks and CD-ROMs to be internally connected to the motherboard and perform basic input/output functions

PATA is an acronym for Parallel Advanced Technology Attachment, often shortened to Parallel ATA. Originally known as IDE (Integrated Drive Electronics) it was the connection of choice not only for hard disks, but for floppy and optical (CD/DVD) disk drives as well.

Parallel refers to the fact that is sent 16 bits at a time through a single 16-bit connection, which is used for data traveling in both directions.Current PATA interfaces can transfer at up to 133 megabytes per second. Most popular interface for hard disk drives and CD drives in mainstream systems for many years.PATA is Half Duplex, meaning transmit and receive cannot occur at the same time.

SATA is an acronym for Serial Advanced Technology Attachment. SATA is a new method of transferring data between the drive and the motherboard (or interface board) that uses a high-speed serial bus instead of a lower-speed parallel one. Another feature you get with SATA is the ability to remove or attach drives without having to shut the computer down. SATA interfaces can transmit up to 150, 300, or 600, 3000 megabytes per second. SATA is full duplex,

 

 

Difference:

1. SATA is the current standard while PATA is already obsolete.
2.SATA is much faster than PATA.
3.SATA is equipped with NCQ while PATA isn’t.(Native Command Queuing (NCQ) is an extension of the Serial ATA protocol allowing hard disk to internally optimize the order in which received read and write commands are executed).
4.SATA drives are hot pluggable while PATA drives are not.
5.SATA provides an external interface while PATA does not.
6.SATA uses smaller cables than PATA.
7.PATA uses a slave/master arrangement while SATA does not.
