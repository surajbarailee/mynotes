File Systems

A file is a resource holding data in a storage device that can be a text,image,video,etc. A file has attributes such as name,identifier(filename for the system that could be number used to uniquely identify a file in a file system),type,loaction,size,access property(read only,read/write,etc).

File system a part of an operating system that manages and keeps records of files and its details. Some types of file system are FAT file system NTFS, file system,network file system,etc.

File system are basically a tool to have an easy view of data ,files and folders stored in the memory and perform necessary operation as and when required.Some of those operations are:

Creating a new file

A new file contains data as like its name ,location,type,etc.which we also refer as metadata.

Reading/Writing a file

For reading /writing a file, the system has to specify name and location (content in writing).Read /Write pointers helps in these kinds of processes.

Copy/Move a file

In most cases copying is done byte by byte to a new target space specified.And moving files normally doesnt require any actual input/ouput operations.

File Access Methods

The information in a file can be accessed in various ways. The most common among them are using sequential access or direct access. More details about these are:

  • Sequential Access
  • Direct Access

 

Sequential Access

The information in a file is processed in order using sequential access. The files records are accessed on after another. Most systems such as editors, compilers etc. use sequential access. It is based on the tape model of a file and so can be used with sequential access devices as well as random access devices.The read and write operations in the file can only be done in a sequential manner. However, the file can be reset to the beginning or rewinded as required.

Direct Access

In direct access also called relative access files can be accessed in random for read and write operations. The direct access model is based on the disk model of a file, since it allows random accesses. In this method, the file is divided into numbered blocks. Any of these arbitrary blocks can be read or written. For example, we may read block 8, then write into block 10 and then read block 15. Direct access system is quite useful and mostly databases are of this type.
