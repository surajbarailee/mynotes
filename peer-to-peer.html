Peer to Peer Architecture

In a peer to peer network,tasks are allocated to every device on the network.

There is no any central server or repository.All computers are considered equal.Instead of a certain point of centralized server,each computer is a server for the files stored in it.

Advantages of a peer-to-peer network:

  • Cost of a powerful centralized server is reduced.
  • No single point of failure.It means that if one computer/node fails other can work seperately (given that the work is isolated from failed computer).
  • Installation and setup becomes a little easy and less knowledge is required.

Disadvantages of a peer to peer network:

  • Data backup if required has to be setup to each computer.
  • More has to be done to make all the computers secure.
  • Installation of softwares like OS has to be installed in all the computers ,thus more time and more money is spent.
